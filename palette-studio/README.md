# Palette Studio

AI-powered color palette generator — Month 2 of BootcampWithAI.

## Running locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Stack

- React 18 + Vite
- CSS Modules (no build-time Tailwind needed)
- Claude API (Month 3)
- Vercel (deployed)

## What's built so far

- [x] Project scaffold (Vite + React)
- [x] ColorCard component — swatch, name, role, copy-to-clipboard
- [x] Seed palette rendered from state
- [x] Prompt input UI (placeholder — AI integration in Month 3)
- [ ] Claude API integration
